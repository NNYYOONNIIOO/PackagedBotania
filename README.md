# PackagedBotania

**PackagedBotania** is a mod that allows automated processing of Botania through packed recipes.

## Description

It includes the Packaged Mana Pool, Packaged Petal Apothecary, Packaged Runic Altar, Packaged Alfheim Portal, and Packaged Terrestrial Agglomeration Plate.

## Features

- **Packaged Mana Pool** – Functions the same as a Mana Pool and can be affected by blocks placed beneath it.
- **Packaged Petal Apothecary** – Functions the same as the Petal Apothecary, but requires an external water supply (e.g., via pipes).
- **Packaged Runic Altar** – Functions the same as the Runic Altar, but can actively draw mana from Sparks.
- **Packaged Alfheim Portal** – Functions the same as the Alfheim Portal, but can actively draw mana from Sparks.
- **Packaged Terrestrial Agglomeration Plate** – Functions the same as the Terrestrial Agglomeration Plate, requires a multiblock structure, and can actively draw mana from Sparks.

The Encoder also has corresponding recipes added for these.

## Integration

- Supports importing recipes from JEI. When importing Petal Apothecary or Runic Altar recipes, Living Rock and Seeds are automatically filled in (configurable).
- Supports Botania Tweaks' modifications to Terrestrial Agglomeration Plate recipes, including new inputs, outputs, and block replacements upon completion.
